# JetHQ Brand

A single-page brand site that visualizes and embeds the canonical JetHQ brand prompt — charcoal ground, warm gold identity, Mrs Eaves serif, Gotham sans, a single topographic motif, and a teal accent reserved for state.

The site demonstrates the brand by living it: rendered lockup, gold rules, type scale, color swatches with HEX/HSL/role, decorative motifs, components (buttons, status, quote tables), and house rules. The full `brand-prompt.md` is embedded at the bottom, ready to hand to any AI tool or designer producing a JetHQ artifact.

## Files

- `index.html` — the brand site (static, no build step)
- `brand-prompt.md` — canonical JetHQ brand prompt, embedded into the page and downloadable
